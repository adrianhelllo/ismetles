Programozás

A nyáron rengeteget programoztam, részben unalmam, másrészt érdeklődésem miatt. A nyár elején végigcsináltam egy kezdő webfejlesztés kurzust, amelyben tovább néztem a CSS és a HTML érdekes funkcióit és trükkjeit. Ezen kívül sokat Pythonoztam, és megépítettem egy Tetris játékot kb. egy hét alatt. Belevágtam a Harvard által mindenkinek elérhető CS50x kurzusba is, melyben C nyelvet kezdtem el tanulni.
